A lightweight lead management plugin with AJAX form, REST API, admin widget, and secure frontend CRUD integration.

== Description ==

Lead Manager helps small businesses and agencies capture and manage leads directly in WordPress.

**Features:**
* Custom Post Types for Leads and Clients
* AJAX-based lead submission form ([lead_form] shortcode)
* Secure Nonce & Capability Checks
* REST API endpoint for external integrations
* Admin dashboard widget for recent leads
* Frontend management page for editing, converting, and deleting leads

== Installation ==
1. Upload `lead-manager-by-jakob-varghese` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The "Submit Lead" page will be automatically created.
4. Optionally create a "Manage Leads" page using the "Edit Leads (Frontend)" template (provided in the Portfolio Portal theme).
